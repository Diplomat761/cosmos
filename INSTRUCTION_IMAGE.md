# Инструкция по добавлению изображения

## Для демонстрации варианта 5 в арабском стиле

1. **Добавьте изображение апельсинового дерева** в папку `images/`
2. **Назовите файл:** `orange-tree.jpg` (или измените путь в CSS)
3. **Рекомендуемый формат:** JPG, PNG
4. **Рекомендуемый размер:** минимум 1920x1080px для качественного отображения

### Альтернатива

Если у вас нет изображения, вы можете:
- Использовать любое другое изображение природы
- Изменить путь в CSS (строка 17 файла `news-variants-demo.html`)
- Или использовать градиентный фон вместо изображения

### Изменение пути изображения

В файле `news-variants-demo.html` найдите строку:
```css
background-image: url('images/orange-tree.jpg');
```

И замените на путь к вашему изображению.
