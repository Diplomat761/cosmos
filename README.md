# Сайт ЖСК "Космос"

Официальный сайт Жилищно-строительного кооператива "Космос" для жителей 80-квартирного дома.

## Структура проекта

```
jsk-kosmos/
├── index.html              # Главная страница
├── meters.html             # Передача показаний счетчиков
├── requests.html           # Подача заявок
├── css/
│   ├── variables.css      # CSS переменные
│   ├── style.css          # Основные стили
│   └── responsive.css     # Адаптивные стили
├── js/
│   ├── main.js            # Основной функционал (меню, прокрутка)
│   ├── meters.js          # Логика передачи показаний
│   └── requests.js        # Логика подачи заявок
├── php/
│   ├── submit_meters.php  # Обработка показаний (заглушка)
│   └── submit_request.php # Обработка заявок (заглушка)
├── images/                # Изображения
└── assets/                # Иконки, документы, фавикон
```

## Функционал

### Главная страница (index.html)
- Информация о ЖСК "Космос"
- Блоки-ссылки на основные функции
- Контактная информация
- Новостная лента
- Адаптивное меню

### Передача показаний (meters.html)
- Форма передачи показаний счетчиков
- Валидация данных
- Расчет потребления и суммы
- История передач (localStorage)
- Автоподстановка предыдущих показаний

### Подача заявок (requests.html)
- Форма подачи заявки с выбором типа
- Загрузка фотографий
- Маска для телефона
- Счетчик символов
- История заявок (localStorage)
- Генерация номера заявки

## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- PHP (заглушки для обработки форм)
- Font Awesome (иконки)
- Google Fonts (Roboto)

## Адаптивность

- Mobile First подход
- Брейкпоинты: 320px, 768px, 1024px, 1280px
- Кроссбраузерность: Chrome, Firefox, Safari, Edge

## Запуск проекта

1. Откройте `index.html` в браузере
2. Для работы PHP заглушек необходим веб-сервер (Apache/Nginx с PHP)

## Особенности

- Все данные сохраняются в localStorage браузера
- Формы валидируются на стороне клиента
- Адаптивное мобильное меню
- Плавная прокрутка к якорным ссылкам
- Модальные окна для подтверждений
